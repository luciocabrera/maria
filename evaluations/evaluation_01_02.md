# 🧪 EXAMEN PRÁCTICO FINAL
🎯 Objetivo

Evaluar si el estudiante puede aplicar lo aprendido en un entorno real.

🥷 Desafío: “Tu Primer Proyecto Ninja”
📌 Descripción

El estudiante debe crear un proyecto desde cero utilizando GitHub y VS Code.

🛠️ Instrucciones Paso a Paso
## 🔹 Parte 1 — Entorno

- Crear carpeta llamada:

mi-dojo-final
Abrirla en VS Code

- Crear archivo:

index.html
## 🔹 Parte 2 — VS Code

El estudiante debe:

Usar terminal para navegar
Instalar mínimo 3 extensiones:
- Live Server
- Prettier
- ESLint
- Activar formateo automático
- ersonalizar tema


## 🔹 Parte 3 — HTML básico

- Crear un archivo index.html con:

  - Título
Nombre del estudiante
  - Anime favorito
Un párrafo de presentación

##  Parte 4 — Git y GitHub

- Inicializar repositorio:

Hacer mínimo 3 commits:
Commit 1: estructura inicial
Commit 2: contenido agregado
Commit 3: mejoras

- Crear repositorio en GitHub llamado:

mi-proyecto-ninja

- Subir el proyecto:

git remote add origin URL
git push -u origin main
# 🔹 Parte 5 — Ramas

- Crear rama:

  - feature/mejora-html
  - Agregar mejora (nuevo contenido o estilo)
  - Hacer merge a main
